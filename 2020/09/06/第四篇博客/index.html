在MATLAB中使用Python

在B站上看到了Python与MATLAB交互的视频，记录一下方法。

总的来说，matlab中直接在前面加上py.就可以调用Python的所有用法。

在MATLAB中使用Python的module

当写好一个Python脚本之后，想要在MATLAB里运行有几种方法

1.直接导入：

​ mod = py.importlib.import_module(module名)

2.重新导入：

​ py.importlib.reload(module名)

3.将当前文件夹加入到Python搜索路径：

​ if count(py.sys.path,’’) == 0

​ insert(py.sys.path,int32(0),’’);

​ end

当Python脚本经过修改之后要重新导入才能执行修改后的脚本。

这个可能存在版本的限制，老版本的MATLAB不一定能实现。
